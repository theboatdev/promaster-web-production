Premium Brand System

Built for Protection. Engineered for Performance.

ProMaster® is a bold construction chemicals brand built around industrial confidence, technical reliability, and premium UAE project performance.

View GuidelinesPackaging System
01 / Brand Foundation

Industrial. Premium. Dependable.

The brand should feel strong, clean, technical and built for harsh construction environments without looking too heavy or outdated.

Positioning

Advanced construction chemical solutions for waterproofing, repair, bonding, coatings and long-term structural protection.

Personality

Professional, durable, performance-focused, confident, modern industrial and technically reliable.

Tone

Clear, direct, premium and technical. Short statements. Strong claims. No unnecessary wording.

02 / Color System

White space. Jet black. Command red.

The palette should blend with the clean logo background while using red only as a high-value action and emphasis color.

Primary White#F8F8F8
Accent Red#C80000
Jet Black#050505
Industrial Silver#D7D7D7
Primary Font
Garret

Use Garret for headlines, product names, hero titles, brochure covers and high-impact brand statements. Use uppercase for maximum authority.

Secondary Font
Poppins

Use Poppins for body text, buttons, technical descriptions, specifications, tables, labels, captions and website interface copy.

03 / Signature Black Line Language

The visual code of ProMaster.

This black angular line system should become a recurring brand asset across packaging, web UI, brochure dividers, social posts and technical documents.

Use it for

  • Product bucket wraps
  • Section dividers
  • Website hero accents
  • Brochure page headers
  • Technical document covers

Keep it

  • Black only for premium consistency
  • Thick and confident
  • Angular and sharp
  • Clean with enough white space
  • Consistent across all products

05 / Packaging System

A consistent product family.

All containers should follow a clean white base, strong black line wrap, red highlights, clear technical label grid and unified hazard / storage / product information structure.

Packaging Rules

  • White base container
  • Black thick wrap design
  • Red only for highlights
  • Clear product hierarchy

Visual Goal

  • Premium industrial
  • Supplier-ready
  • Recognizable on shelves
  • Consistent across all sizes
